# Saathi Community Clinic Health Information System Design

## 📌 Objective
Designed a Health Information System (HIS) for a community clinic to support patient care, telehealth services, and efficient data management.

## 🏥 Overview
This project presents a centralized EHR-based system integrating scheduling, billing, lab systems, telehealth, and patient portals.

## 🔗 Key Features
- Central Electronic Health Record (EHR)  
- Telehealth and remote monitoring  
- Patient portal  
- Billing and claims system  
- Lab and pharmacy integration  

## 🔄 Interoperability
- HL7 and FHIR standards  
- Data exchange with external systems  

## 🔐 Security & Privacy
- HIPAA compliance  
- Role-based access  
- Data encryption  

## 🎯 Outcome & Contribution
- Designed EHR-based system architecture  
- Contributed to privacy and security planning  
- Demonstrated understanding of healthcare workflows and interoperability
